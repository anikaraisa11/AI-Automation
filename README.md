
# 📈 AI-Powered Stock Analysis Telegram Agent
An automated financial assistant built with n8n that generates real-time technical analysis charts and AI-driven market insights directly within Telegram.

# 🌟 Overview
This project leverages a multi-workflow architecture to bridge the gap between market data and artificial intelligence. By simply messaging a ticker symbol to a Telegram bot, users receive a professional-grade technical chart (MACD, Volume, Candlestick patterns) followed by a deep-dive analysis generated by Google Gemini.

# ⚙️ How It Works
The system operates using two primary layers:

The Controller (Agent Workflow): A Telegram Trigger receives the ticker. The AI Agent uses a "Call Workflow" tool to fetch data, then summarizes the findings for the user.

The Engine (Sub-workflow): This workflow handles the heavy lifting—requesting the chart from the Chart-Img API, downloading the image, and using an image-to-text model to perform "Visual" technical analysis.

# 🛠 Tech Stack
Automation: n8n (Low-code workflow engine)

AI Model: Google Gemini (via Gemini Chat Model node)

Messaging: Telegram Bot API

Data/Charts: Chart-Img API (TradingView integration)

Database/Memory: n8n Simple Memory (to maintain conversation context)

# 🚀 Key Features
Visual Chart Generation: Automatically generates specialized charts with MACD and Volume overlays.

Multimodal Analysis: Uses AI to "look" at the chart image and describe candlestick patterns (like Bullish Engulfing) and resistance zones.

Conversational Memory: Remembers previous tickers so you can ask follow-up questions about the same stock.

Sub-workflow Modularization: The stock-fetching logic is separated from the chat logic, making it easy to upgrade or reuse.

# 📥 Setup Instructions
Telegram: Create a bot via @BotFather and copy the API Token.

Chart-Img API: Sign up at chart-img.com and get your API Key.

n8n Credentials: * Add Telegram API credentials.

Add Google Gemini API credentials.

Configure the HTTP Request node with your Chart-Img Authorization header (Bearer YOUR_TOKEN).

Workflow Configuration:

Import the Sub-workflow first and ensure the "Workflow Input Trigger" is set.

Import the Main Agent workflow and link the "Call n8n Workflow" tool to your Sub-workflow.

Click Publish on both workflows.

# 🕹 How To Use
Open your bot in Telegram.

Send a message like: "Analyze AAPL" or "What's the outlook for BTC?"

Wait a few seconds for the agent to:

Fetch the latest weekly/daily chart.

Analyze the MACD and price action.

Send you the image and a text breakdown.
