# 🚀 AI-Powered Lead Generation & Enrichment Agent
An automated, full-stack lead generation workflow built with n8n. This system captures user search queries via webhooks, performs deep web scraping using Apify, processes the data through a Google Gemini-powered AI Agent, and organizes the final leads into Google Sheets.

# 🛠️ Tech Stack
Workflow Automation: n8n

Data Extraction: Apify (Web Scraping Actors)

Intelligence Layer: Google Gemini Chat Model (LLM)

Storage: Google Sheets

Communication: Webhook (REST API)

# 🏗️ Workflow Architecture
The workflow follows a linear, high-integrity path from raw input to structured output:

Webhook Trigger: Receives a POST request from the frontend containing the search_query, location, and limit.

Apify Integration (Run Actor): Triggers a specific Apify actor to scrape real-time data based on the user's parameters.

Data Retrieval: Fetches the resulting dataset items once the scraping task is complete.

Data Filtering: Cleanse the dataset to ensure only high-quality, relevant leads pass through the pipeline.

Intelligent Processing (The AI Agent): * Iterates through each lead using a Loop.

Uses Google Gemini to analyze lead details, categorize the business, or generate a personalized outreach strategy.

Data Persistence: Appends the enriched data into a dedicated Google Sheet for immediate sales use.

# 🌟 Key Features
Agentic Logic: Uses an AI Agent node rather than a standard completion node, allowing for more nuanced data parsing and decision-making.

Scalable Scraping: Offloads heavy-duty web crawling to Apify, ensuring the workflow remains fast and avoids IP blocking.

Real-time Processing: Designed to handle incoming leads instantly as users interact with the frontend.

Error Handling: Includes filtering stages to prevent empty or corrupted data from reaching the final database.

# 🚀 Getting Started
Prerequisites
An active n8n instance.

Apify API Key and a configured Actor (e.g., Google Maps Scraper or LinkedIn Scraper).

Google Gemini API Key.

A Google Cloud Project with the Google Sheets API enabled.

Installation
Import Workflow: Copy the JSON export of this workflow and paste it into your n8n canvas.

Configure Credentials:

Set up your Apify API credentials.

Connect your Google Gemini Chat Model node.

Authorize the Google Sheets node.

Update Webhook: Replace the webhook URL in your frontend code with the production URL generated by n8n.

Execute: Hit "Execute Workflow" to test the end-to-end integration.
